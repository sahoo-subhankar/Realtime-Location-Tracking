# Realtime-Tracking
Realtime Location Tracking System

### Introduction
Track the Real-time location just like: Uber, Rapido, Swiggy, Zomato etc. apps does.




### Requirements
- `Javascript`
- `Node.js`
- `Express.js`
- `Socket.io`
- `ejs`
